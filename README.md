# Čechlo Inventory

Jednoduchá Next.js + Supabase appka pro správu zákazníků, leadů a vozíků.

## Lokální spuštění

1. Nainstaluj závislosti:
   ```bash
   npm install
   ```
2. Zkopíruj `.env.example` na `.env.local` a doplň hodnoty z Supabase.
3. Spusť vývojový server:
   ```bash
   npm run dev
   ```
4. Otevři `http://localhost:3000`.

## Supabase schéma

V SQL editoru Supabase spusť skript z `supabase/schema.sql`.
